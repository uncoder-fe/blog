# 哈哈哈哈

- 函数式编程(Functional Programming)

- 数据流编程(Dataflow Programming)

- 面向对象编程(Object Oriented Programming )

# 然而

上面的术语，是有几个大类分的

- 声明式（包含函数式，数据流编程）
- vs
- 命令式(或指令式)编程(包含面向对象，面向过程)

- <--分割线-->

- 结构化
- vs
- 非结构化
